editorialDetailCtrl
controller in module editorials

Description

Definición de un controlador auxiliar del módulo Editorial. Se crea el controlador con el cual se manejan las vistas de detalle del módulo.

Usage

editorialDetailCtrl($scope, $http, editorialContext, $state);

Parameters

ParamTypeDetails
$scopeObject

Referencia injectada al Scope definida para este controlador, el scope es el objeto que contiene las variables o funciones que se definen en este controlador y que son utilizadas desde el HTML.

$httpObject

Objeto injectado para la manejar consultas HTTP

editorialContextObject

Constante injectada que contiene la ruta donde se encuentra el API de Editoriales en el Backend.

$stateObject

Dependencia injectada en la que se recibe el estado actual de la navegación definida en el módulo.

Methods

  • getEditorialID(URL)

    Esta función utiliza el protocolo HTTP para obtener el recurso donde se encuentra la editorial por ID en formato JSON.

    Parameters
    ParamTypeDetails
    URLString

    Dirección donde se encuentra el recurso de la editorial o API donde se puede consultar.
