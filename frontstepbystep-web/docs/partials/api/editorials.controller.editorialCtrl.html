editorialCtrl
controller in module editorials

Description

Definición del controlador de Angular del módulo Editorial. Se crea el controlador con el cual se maneja el módulo. En el controlador se definen los atributos y métodos que pueden ser accedidos desde el HTML utilizando el $scope.

Usage

editorialCtrl($scope, $http, editorialContext);

Parameters

ParamTypeDetails
$scopeObject

Referencia injectada al Scope definida para este controlador, el scope es el objeto que contiene las variables o funciones que se definen en este controlador y que son utilizadas desde el HTML.

$httpObject

Objeto injectado para la manejar consultas HTTP

editorialContextObject

Constante injectada que contiene la ruta donde se encuentra el API de Editoriales en el Backend.

Methods

  • getEditorials(URL)

    Esta función utiliza el protocolo HTTP para obtener el recurso donde se encuentran las editoriales en formato JSON. El recurso puede ser un archivo o un API Rest. La función se ejecuta automáticamente cuando el controlador es accedido desde el navegador.

    Parameters
    ParamTypeDetails
    URLString

    Dirección donde se encuentra el recurso de las editoriales o API donde se puede consultar.
