logoutCtrl
controller in module login

Description

Definición del controlador de cerrar sesión del módulo de Login. Se crea el controlador con el cual se maneja el módulo. Al cargarse automáticamente cierra la sesión.

Usage

logoutCtrl($rootScope, $state);

Parameters

ParamTypeDetails
$rootScopeObject

Referencia injectada al Scope definido para toda la aplicación.

$stateObject

Dependencia injectada en la que se recibe el estado actual de la navegación definida en el módulo.
