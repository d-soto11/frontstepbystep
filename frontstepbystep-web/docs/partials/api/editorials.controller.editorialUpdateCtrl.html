editorialUpdateCtrl
controller in module editorials

Description

Definición del controlador auxiliar para actualizar Editoriales.

Usage

editorialUpdateCtrl($scope, $http, editorialContext, $state, booksContext);

Parameters

ParamTypeDetails
$scopeObject

Referencia injectada al Scope definida para este controlador, el scope es el objeto que contiene las variables o funciones que se definen en este controlador y que son utilizadas desde el HTML.

$httpObject

Objeto injectado para la manejar consultas HTTP

editorialContextObject

Constante injectada que contiene la ruta donde se encuentra el API de Editoriales en el Backend.

$stateObject

Dependencia injectada en la que se recibe el estado actual de la navegación definida en el módulo.

booksContextObject

Constante injectada que contiene la ruta donde se encuentra el API de Libros en el Backend.

Methods

  • createEditorial(id, editorial)

    Esta función utiliza el protocolo HTTP para actualizar la editorial por ID en formato JSON.

    Parameters
    ParamTypeDetails
    idString

    El ID de la editorial a actualizar.

    editorialObject

    Objeto con la información nueva de la editorial.

  • getEditorialID(URL)

    Esta función utiliza el protocolo HTTP para obtener el recurso donde se encuentra la editorial por ID en formato JSON.

    Parameters
    ParamTypeDetails
    URLString
