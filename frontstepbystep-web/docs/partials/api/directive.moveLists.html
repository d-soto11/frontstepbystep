directive:moveLists

Description

Crea una vista que permite intercambiar elementos entre dos arreglos

Usage

as element:
<move-lists
       selected
       available="{array}">
</move-lists>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 0.

Parameters

ParamTypeDetails
selectedarray

Lista de items seleccionades

availablearray

Lista de items posibles

Example

<move-lists selected="selectedItems" available="availableItems"></move-lists>
