authorCtrl
controller in module authors

Description

Definición del controlador de Angular del módulo Autores. Se crea el controlador con el cual se maneja el módulo. En el controlador se definen los atributos y métodos que pueden ser accedidos desde el HTML utilizando el $scope.

Usage

authorCtrl($scope, $http, authorsContext);

Parameters

ParamTypeDetails
$scopeObject

Referencia injectada al Scope definida para este controlador, el scope es el objeto que contiene las variables o funciones que se definen en este controlador y que son utilizadas desde el HTML.

$httpObject

Objeto injectado para la manejar consultas HTTP

authorsContextObject

Constante injectada que contiene la ruta donde se encuentra el API de Autores en el Backend.

Methods

  • getAuthors(URL)

    Esta función utiliza el protocolo HTTP para obtener el recurso donde se encuentran los autores en formato JSON. El recurso puede ser un archivo o un API Rest. La función se ejecuta automáticamente cuando el controlador es accedido desde el navegador.

    Parameters
    ParamTypeDetails
    URLString

    Dirección donde se encuentra el recurso de los autores o API donde se puede consultar.
